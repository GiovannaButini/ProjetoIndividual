Cores

A ciência que estuda as cores é denominada calorimetria. Este estudo desenvolve métodos de quantificação da cor, além de analisar os tons, as saturações e as intensidades das cores.



O tom da cor é o que faz com que ela seja identificada como azul, verde, vermelho, etc. A saturação mostra se a cor é natural ou pigmentada artificialmente. Já a intensidade caracteriza a força da cor.



Todo esse conjunto das cores, é subdivido em: primárias, secundárias, terciárias ou neutras.



Cores primárias





As cores primárias são: vermelho, azul e amarelo. O vermelho é uma cor quente que transmite ao nosso cérebro vitalidade, energia e coragem. O amarelo é uma cor suave e alegre que simboliza o otimismo. O azul é a cor que dá concentração e melhora a mente.



Cores secundárias





As cores secundárias são duas cores que geram conforme junção de duas cores primárias, como representada na imagem acima.



Cores terciárias





A cores terciárias resultam da mistura de uma cor primária com uma ou duas cores secundárias, ou seja, são todas as outras cores que conhecemos.



Cores neutras





Essas cores são utilizadas para complementar uma cor desejada, sendo que as cores neutras têm pouco reflexo. Entre as cores neutras, pode-se citar o branco, que é uma luz isenta de cor, o preto, que é a ausência de cor, marrom e tons de cinzas que são formados através da mistura de preto e branco.



Cores quentes





As cores quentes são as mais vivas e energéticas.



Cores frias





As cores frias dão a sensação de calma e tranquilidade.



Policromia





É a arte feita com várias cores. É obtida através da junção das três cores primárias mais o preto para realçar.



Monocromia





É uma pintura que emprega vários tons de uma mesma cor. É uma harmonia obtida através da adição gradativa de branco ou preto a uma única cor primária, secundária ou terciária.
